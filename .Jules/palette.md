## 2026-05-11 - [Theme Toggle Accessibility]
**Learning:** Visually hiding custom checkboxes (using opacity: 0, width: 0, height: 0, position: absolute) instead of display: none is crucial to preserve keyboard accessibility and screen reader support, as well as adding :focus-visible outlines on adjacent elements for keyboard navigation.
**Action:** Use visually hidden patterns instead of display: none for custom checkboxes.
