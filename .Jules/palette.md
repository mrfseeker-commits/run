## 2026-03-05 - Custom Toggle Switch Accessibility
**Learning:** Using `display: none` on `<input type="checkbox">` elements completely hides them from screen readers and prevents them from receiving keyboard focus. This breaks accessibility for custom toggle switches. Additionally, `.focus()` in Playwright doesn't trigger the `:focus-visible` pseudo-class correctly.
**Action:** Use the visually hidden pattern (`position: absolute; opacity: 0; width: 0; height: 0;`) instead of `display: none` for the hidden input. Apply `:focus-visible` outline to the adjacent visible element (`input:focus-visible + .slider { ... }`) to provide clear visual feedback during keyboard navigation. To test `:focus-visible` in Playwright, simulate an actual "Tab" keypress (`page.keyboard.press("Tab")`).
